# Basic Calculator using Python and CustomTkinter
A simple calculator application built with Python and the CustomTkinter library.
This project serves as an example of how to create a graphical user interface (GUI) application in Python, 
allowing users to perform basic arithmetic operations like addition, subtraction, multiplication, and division.
- Supports basic arithmetic operations: addition, subtraction, multiplication, and division.
- User-friendly graphical interface.
